# E-Commerce-SQL-Data-Analysis
SQL-based analysis of the "Olist E-Commerce dataset" to evaluate order fulfilment, customer satisfaction, seller performance, and payment behaviour.

📊 E-Commerce SQL Data Analysis

📌 Project Overview :

This project analyzes the Brazilian Olist E-Commerce dataset using SQL Server. The objective is to evaluate order fulfillment, customer satisfaction, seller performance, product performance, and payment behavior to generate actionable business insights.

----------------------------------------------------------------------------------------------------------------------------------------------------------------
Problem Statement :

An e-commerce company wants to understand customer purchasing behavior, delivery performance, and seller efficiency to improve business operations and customer satisfaction.

This project answers 17 business research questions using SQL.

Key Insights
- More than 97% of orders were successfully delivered.
- Credit Card was the most frequently used payment method.
- Some sellers consistently required longer delivery times.
- Delivery delays were associated with lower customer review scores.
- Top product categories generated the highest order volumes.
